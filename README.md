# Discord Bot

Tento bot je vytvořen pomocí `discord.py` a nabízí širokou škálu příkazů pro zábavu, moderaci a informace.  

## 🚀 Funkce

- 🔍 **Info příkazy** – `help`, `info`, `userinfo`, `serverinfo`
- 🎮 **Minecraft příkazy** – `mcplayerinfo`, `mcserver`, `achievement`, `mchead`, `item`, `mctotem`, `mcbust`, `mchelm`
- 🤖 **Obrázkové příkazy** – `meme`, `wanted`, `jail`, `blur`, `invert`, `greyscale`, `clown`, `colorify`, `nokia`, `communism`
- 🔮 **Zábavné příkazy** – `8ball`, `gayrate`, `simprate`, `iq`, `sigmarate`, `skibidirate`, `nitro`, `pesti_dostanes`
- 🌍 **Užitečné příkazy** – `translate`, `weather`, `alert`, `texttomorse`, `reverse`, `doublestruck`, `caution`, `encode`, `decode`
- ⚙️ **Moderace** – `embed`, `clear`
- 🐶 **Zábava s mazlíčky** – `pet`
- 🎉 **Další** – `welcome_card`, `startgen`, `stopgen`
